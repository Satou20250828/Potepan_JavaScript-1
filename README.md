# ストップウォッチアプリ（JavaScript）

## 概要
プログラミングスクールの実践課題として、  
JavaScriptの基礎文法とイベント処理を学ぶ目的で作成した、  
ブラウザ上で動作するストップウォッチアプリです。  
スタート・ストップ・リセット操作に応じて、経過時間を表示します。

## 作成した目的
本アプリは、JavaScriptにおける時間処理やイベント処理の仕組みを理解することを目的として作成しました。  
特に、`setInterval` を用いた定期処理やDOM操作を通じて、  
ユーザー操作と画面表示がどのように連動するかを学ぶことを意識しました。

## 使用技術
- HTML  
- CSS  
- JavaScript  

## 実装した機能
- スタート機能  
- ストップ機能  
- リセット機能  
- 経過時間の表示  
- ボタンクリックによるイベント処理  

## 工夫した点
- スタートボタンを連打しても、  
  処理が重複して実行されないよう制御しました。
- 時間の表示形式を整え、  
  ユーザーが経過時間を直感的に把握できるようにしました。
- シンプルで操作しやすいUIを意識しました。

## 苦労した点
0.1秒単位で時間を表示するストップウォッチを実装した際、  
内部で計測している時間と、画面に表示される数値がずれてしまう不具合が発生しました。

原因を確認したところ、  
計測用の時間管理と表示更新のタイミングが一致していないことが分かりました。  
そこで、以下のように実装を見直しました。

- 計測用の時間をミリ秒単位で管理する  
- 表示用の数値は、その計測値から毎回計算して表示する  

この修正により、  
計測時刻と表示が常に一致するようになりました。

この経験から、  
**非同期処理や時間処理では、内部の状態を正しく管理することが重要である**  
ということを学びました。
